## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lachholland/multi-stage-recommender.git
   ```
3. Run the makefile in the project root
   ```sh
   make
   ```
4. Train the model by running the following command in the /src folder
   ```sh
   python -m main
   ```
